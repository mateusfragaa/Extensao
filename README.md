# Projeto de Extensão - ADS (ERP de Pequeno Porte) 🚀

Este projeto consiste no desenvolvimento de um **ERP online de pequeno porte** focado em microempreendedores, varejistas e autônomos.  
O objetivo é **otimizar processos gerenciais** e substituir o uso de planilhas por uma ferramenta tecnológica eficiente.  

Este é um **projeto de extensão**, desenvolvido para **auxiliar a comunidade** local, oferecendo uma solução prática para negócios de pequeno porte. O projeto também serve como campo de testes para o **micro framework PHP FasMicro** e é realizado no contexto da **FAMS**.

## 📋 Funcionalidades Principais
- **Gestão de Estoque:** CRUD de produtos e controle de inventário.  
- **Vendas:** Registro de saídas de produtos.  
- **Financeiro:** Controle básico de contas a pagar e a receber.  
- **Áreas Distintas:** Área pública de apresentação e área administrativa restrita via login.
