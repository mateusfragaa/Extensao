# Projeto de Extensão - ADS (ERP de Pequeno Porte) 🚀

Este projeto consiste no desenvolvimento de um **ERP online de pequeno porte** focado em microempreendedores, varejistas e autônomos, com o objetivo de otimizar processos gerenciais e substituir o uso de planilhas por uma ferramenta tecnológica eficiente.  
O projeto é desenvolvido no contexto da **FAMS - Faculdade de Administração e Sistemas** e serve também como campo de testes para o **micro framework PHP FasMicro**.

## 📋 Funcionalidades Principais
- **Gestão de Estoque:** CRUD de produtos e controle de inventário.  
- **Vendas:** Registro de saídas de produtos.  
- **Financeiro:** Controle básico de contas a pagar e a receber.  
- **Áreas Distintas:** Área pública de apresentação e área administrativa restrita via login.
